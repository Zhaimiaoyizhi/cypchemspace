化学结构与 fingerprint 模块需求

职责：用 RDKit 解析标准化 SMILES，并生成 Morgan fingerprint matrix。

  • API：mol_from_smiles, canonicalize_smiles, make_morgan_fingerprint_matrix。
  • 无效 SMILES 不应导致整个流程崩溃，应被排除或返回 None。
  • 默认 fingerprint 为 Morgan radius 2, 2048 bits；CLI 可调整 bits 和 radius。
  • 验收：测试覆盖无效 SMILES、等价 SMILES 规范化、矩阵维度。
